## Deployment to GitHub Pages

### Option 1: Using the root directory
1. Create a new GitHub repository
2. Copy the contents of `/app/frontend/public/` to your repository root:
   - `index.html`
   - `style.css`
   - `script.js`
3. Push to GitHub
4. Go to Settings > Pages
5. Select "Deploy from branch" and choose `main` branch and `/ (root)` folder
6. Your site will be live at `https://yourusername.github.io/repository-name/`

### Option 2: Using the docs folder
1. Create a new GitHub repository
2. Create a `docs` folder in your repository
3. Copy all files from `/app/frontend/public/` to the `docs` folder
4. Push to GitHub
5. Go to Settings > Pages
6. Select "Deploy from branch" and choose `main` branch and `/docs` folder
7. Your site will be live at `https://yourusername.github.io/repository-name/`
